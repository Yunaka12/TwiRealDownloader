TwiRealDownloader
ver1はエラーの発生がとても多くて使い物になりません。ver2の方がよく動きます。

APP_KEYおよびAPP_SECRETはhttps://apps.twitter.com/から各自用意する必要があります。
また、プログラム上で使用されているモジュールのインポートも必要です。

このプログラムを起動している間に特定ユーザーのツイートに更新があった場合、そのデータ（ツイートおよび添付メディア）を保存します。

このプログラムを起動すると、まず、カレントディレクトリ内に「特定ユーザーのID」フォルダが作られます。

プログラムが起動されている間はずっと、特定ユーザーのツイートの更新を監視します。
更新があった場合、上記フォルダにテキストファイルを作り、そのファイルにツイートとツイート日時を書きこんでいきます。

ただし、ツイート削除も更新とみなされてしまうので注意してください。
なお、ツイートに絵文字が含まれている場合は自動的に除かれます。しかし、除ききれない絵文字があり、その場合はツイートの取得に失敗します。

また、ツイートに動画、GIF、画像が添付されていた場合は、そのメディアも自動的に「特定ユーザーのID」フォルダに保存されます。
それぞれのメディアについて、画像は.jpg、動画は.mp4、GIFは拡張子.mp4として保存されます。
